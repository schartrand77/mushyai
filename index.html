Build a Meshy-style 3D generation experience

A layout inspired by the shared Gemini conversation: from prompt and image intake to reconstruction, texturing, and export.

1. Input Studio

Collect prompts or concept images and prepare them for reconstruction.

  • Text prompt composer
  • Image upload & background removal
  • Style and polycount presets

2. LRM Reconstruction

Generate a coarse geometry pass with Large Reconstruction Models.

  • Single-view / multi-view mode
  • Depth + normal estimation
  • Progressive mesh refinement

3. Texture & Material Pass

Apply physically based materials to produce game-ready outputs.

  • PBR texture generation
  • UV unwrapping and seam cleanup
  • Smart relighting previews

4. Export & Delivery

Ship assets directly into creator workflows.

  • GLB / FBX / OBJ export
  • Auto LOD and decimation
  • API and webhook callbacks
